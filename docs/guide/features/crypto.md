# Crypto

Real-time cryptocurrency prices from Binance.

## Supported Coins

30+ cryptocurrency pairs including BTC, ETH, SOL, DOGE, XRP, ADA, AVAX, and more.

## Features

- Real-time price updates via WebSocket
- 24-hour change percentage
- Volume data
- Exchange source badge
- Smart price precision (2 decimals for BTC, up to 8 for SHIB)

## Crypto Detail Page

Click any coin to view `/crypto/[symbol]` with:
- Live price with change indicator
- Candlestick chart with time range selection
- Market data panel (bid, ask, spread, volume)
