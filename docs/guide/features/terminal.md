# Terminal

A fully customizable workspace with drag-and-drop widgets.

## Widget Types

| Widget | Description |
|--------|-------------|
| Chart | Full candlestick chart with interval/range selection |
| Mini Chart | Compact area chart |
| Watchlist | Live price feed from your watchlist |
| News Feed | Market news stream |
| Portfolio | Holdings with live P&L |
| Heatmap | Custom stock/currency heatmap |
| Bank Rates | Turkish bank buy/sell/spread comparison |

## Pages

Three switchable terminal pages (T1, T2, T3). Each page has its own independent widget layout. Widget positions and configurations are saved per user.

## Interactions

- **Drag**: Move widgets by grabbing the title bar
- **Resize**: Drag the bottom-right corner handle
- **Add**: Click "Add Widget" or right-click the canvas
- **Remove**: Click the X button on the widget title bar
