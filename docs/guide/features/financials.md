# Financials

Company-level financial analysis powered by Finnhub.

## Features

- **Earnings Calendar**: Upcoming and past earnings events with actual vs. estimate
- **Company Metrics**: Financial ratios (P/E, EPS, market cap, beta, etc.)
- **Earnings History**: Beat/miss visualization with bar charts
- **Analyst Recommendations**: Strong Buy through Strong Sell breakdown
- **Earnings Quality**: Growth, profitability, leverage, and cash generation scores
- **Price Chart**: Historical price chart for the selected symbol
