# Currencies

Real-time Turkish gold, silver, and forex prices from doviz.com.

## Categories

- **Gold**: 17 variants (Gram, Ceyrek, Yarim, Tam, Cumhuriyet, Ata, Resat, Ons, etc.)
- **Silver**: USD/oz spot price
- **Precious Metals**: Platinum, Palladium
- **Currency**: USD, EUR, GBP
- **Index**: BIST 100

## Bank Comparison

Each currency shows buy/sell prices from 38+ Turkish banks with spread and margin calculations. Sort by sell price, buy price, spread, spread %, or change %.

## Data Source

Connects to doviz.com via WebSocket for real-time streaming. Data includes bid, ask, daily/weekly/monthly/yearly change percentages.
